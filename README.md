# unlimited-lantern
>你懂的，我不解释

具体使用方法类似当年的珊瑚虫，可以修改PE结构导入，或者由注入器加载，不懂的话可以百度下，这里不做教学。懂的人可以issues发个教学，笑。

`lantern.ini` 为配置设置文件，内有详细注释，5种补丁算法，时间控制等。
